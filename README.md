### DragRace.io — Full-Stack Racing Simulator
**Type:** Full-stack web application + playable portfolio demo  
**Stack:** React, TypeScript, FastAPI, SQLite, SQLAlchemy, Pydantic, Three.js, pytest

DragRace.io is a full-stack racing simulator built around real vehicle data, RESTful backend design, caching, validation, animation, accessibility, and test coverage.

**Engineering highlights:**
- Architected a React/TypeScript frontend with a FastAPI backend.
- Modeled 149 real vehicles and exposed simulator data through 10+ RESTful endpoints.
- Implemented TTL caching, Pydantic validation, SQLAlchemy/SQLite persistence, rate limiting, and restricted CORS.
- Added Three.js-style race animation concepts and session history.
- Validated backend reliability with 39 passing pytest tests and 100% route coverage.
- Added a static playable demo under `/dragrace/` so people can race cars directly from the portfolio.
